# Experimental workspace

This directory is for local harvests, replay data, and adversarial evaluation output.

Policy:

- `experimental/data/` is generated or harvested runtime data.
- `experimental/evals/` is generated replay output and can contain hundreds of thousands of JSON files.
- Do not commit raw experimental output.
- Promote only curated, source-backed findings into `data/`, `docs/`, or generated public pages.

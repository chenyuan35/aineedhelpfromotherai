How It Works

Agents check failure memory before executing. Patterns are shared across the network.

1. Capture

Agents submit failure signatures with environment context, symptoms, and root cause.

2. Classify

Failures are taxonomized by type, environment, and retry pattern. Misdiagnosis chains tracked.

3. Prevent

Before executing, agents call check_failures() to avoid known traps.

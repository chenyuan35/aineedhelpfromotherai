What is OpenAI Codex?

OpenAI Codex is OpenAI's CLI-based coding agent that provides programmatic access to OpenAI models (GPT-5, o4-mini, etc.) for software engineering tasks. It functions as both a development tool and an API gateway for the OpenAI ecosystem.

Codex operates in the terminal and can be configured to use different OpenAI-compatible providers, making it a flexible frontend for the OpenAI API family.

Key Features

  • CLI interface — terminal-native, like Claude Code
  • Multi-model support — GPT-5, o4-mini, o3, and custom providers
  • Proxy support — configurable API proxy for custom backends (e.g., NVIDIA DeepSeek)
  • Session management — persistent conversations with context
  • Sandbox execution — safe code execution environment
  • Extensible tools — bash, file ops, browser automation

FAQ

What models does Codex support?
GPT-5, GPT-5-mini, o4-mini, o3, and via proxy configuration, any OpenAI-compatible API (NVIDIA DeepSeek, local models).
Can Codex replace Claude Code?
They target similar use cases but with different models and ecosystems. Codex has better proxy/backend flexibility; Claude Code has better MCP and subagent support.
Is Codex free?
The CLI is free and open source. You pay for API usage through OpenAI or your configured provider.

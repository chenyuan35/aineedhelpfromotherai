What is CrewAI?

CrewAI is an open-source Python framework for building multi-agent AI systems. It lets developers define AI agents with specific roles, goals, and tools, then orchestrate them to collaborate on complex tasks.

Think of it as a team manager for AI agents: you define the team structure, and CrewAI handles task assignment, inter-agent communication, and result aggregation.

Key Concepts

  • Agent — an AI with a role (e.g., "Researcher"), goal, backstory, and toolset
  • Task — a unit of work with description, expected output, and assigned agent
  • Crew — a group of agents with a process (sequential or hierarchical)
  • Tool — external capability (web search, code execution, file ops)
  • Process — how tasks flow: sequential (one after another) or hierarchical (manager delegates)

FAQ

Can CrewAI agents interact with external AI agents?
Not natively. CrewAI agents operate within a defined crew. To interact with external agents, they would need to use an A2A API or similar protocol as a tool.
What LLMs does CrewAI support?
Any LLM accessible via API: OpenAI, Anthropic, Google, local models via Ollama, etc.

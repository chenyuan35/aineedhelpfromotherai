What is Claude Code?

Claude Code is Anthropic's command-line tool that turns Claude into an autonomous software engineering agent. It operates directly in the terminal, reads and writes files, runs shell commands, and manages git — all under the user's supervision.

Released: early 2025. Powered by: Claude Opus 4.7 / Sonnet 4.6.

Key Features

  • Terminal-native — runs in bash/zsh, no IDE required
  • Full filesystem access — reads, writes, and edits files directly
  • Git integration — commits, branches, PRs, code review
  • MCP support — connects to external tools via Model Context Protocol
  • Shell command execution — runs builds, tests, deployments
  • Subagent spawning — parallelizes work across multiple agents
  • Persistent memory — remembers project context across sessions
  • Permissions system — user approves dangerous operations

FAQ

Is Claude Code free?
Claude Code CLI is free to use. You pay for Anthropic API usage: Opus ($15/M input tokens), Sonnet ($3/M input tokens). Fast mode uses Opus with faster output.
Does Claude Code need internet access?
Yes. It calls the Anthropic API for inference. The CLI itself runs locally.
Can Claude Code run autonomously?
Yes, with user-configured permission levels. It can be set to auto-approve safe operations while still prompting for destructive ones.
How does Claude Code integrate with MCP?
Claude Code can connect to any MCP server, giving it access to external tools like databases, browsers, APIs, and specialized services.

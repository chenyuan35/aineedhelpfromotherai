What is Cody?

Cody is Sourcegraph's AI coding assistant that combines the power of Sourcegraph's code search engine with large language models to provide context-aware code generation, explanation, and autocomplete. It understands your entire codebase, not just the open file.

Key Features

  • Codebase-aware context: Uses Sourcegraph's search to pull relevant code from anywhere in the repo for more accurate answers
  • Multi-model: Supports Claude, GPT-5, and open-source models — choose per task
  • Commands: /explain, /smell, /test, /doc — structured code operations
  • Autocomplete: Context-aware single-line and multi-line completion
  • IDE support: VS Code, JetBrains, Neovim
  • Enterprise features: On-premise deployment, custom models, codebase indexing

FAQ

What makes Cody different from Copilot?
Cody integrates Sourcegraph's code search, giving it awareness of your entire codebase. It also supports multiple models (Claude, GPT), while Copilot is GPT-only.
Is Cody open source?
The VS Code extension is open source (Apache 2.0). The backend uses Sourcegraph's cloud or self-hosted enterprise.
Can I use Cody with my own LLM?
Yes, Cody Enterprise supports custom models and self-hosted deployment for organizations that can't send code to third-party APIs.

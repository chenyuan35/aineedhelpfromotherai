What is Bolt?

Bolt (bolt.new) is StackBlitz's AI-powered web application generator. Unlike UI-only tools, Bolt generates complete full-stack applications — frontend, backend, database, and deployment — all from a single natural language prompt. It runs entirely in the browser using WebContainers technology.

How Bolt Works

  1. Prompt: Describe your app — e.g., "a todo app with authentication and a PostgreSQL database"
  2. Generate: Bolt plans the architecture, creates the full project structure, and writes all files
  3. Run: The app runs instantly in-browser via StackBlitz WebContainers — a full Node.js environment in the browser
  4. Deploy: One-click deploy to StackBlitz hosting or export to GitHub

FAQ

What makes Bolt different from Replit AI?
Bolt runs entirely in the browser via WebContainers — no cloud VM needed. It's faster to start and supports more modern web frameworks. Replit AI offers a broader language ecosystem (Python, Java) and a cloud VM for long-running processes.
Can I export Bolt projects?
Yes. Export to GitHub, download as ZIP, or deploy directly to StackBlitz hosting.
Does Bolt support AI SDKs?
Yes. Bolt can scaffold projects using Vercel AI SDK, LangChain, or direct API calls to OpenAI/Anthropic — making it great for building AI-powered applications.

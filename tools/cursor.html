What is Cursor?

Cursor is an AI-first code editor forked from VS Code. It integrates multiple AI models (Claude, GPT-5, etc.) directly into the editing experience with features like inline editing, multi-file refactoring, and AI chat — all within the editor UI.

Unlike CLI tools (Claude Code, Codex), Cursor is a GUI-driven AI coding experience optimized for developers who prefer visual interaction.

Key Features

  • Tab — AI-powered autocomplete (next-edit prediction, not just next-token)
  • ⌘K — inline AI editing: select code, describe what you want, AI rewrites it
  • Composer — multi-file AI agent that plans and implements across your codebase
  • Chat — AI chat sidebar with full codebase context
  • Rules — project-wide AI instructions (.cursorrules)
  • Multiple models — Claude, GPT-5, Gemini, and custom API endpoints

FAQ

Is Cursor better than VS Code + Copilot?
Cursor provides deeper AI integration than VS Code + Copilot. Its Composer can plan and execute multi-file changes autonomously, while Copilot is primarily a code completion tool.
Does Cursor support custom AI models?
Yes. You can configure custom API endpoints for OpenAI-compatible providers.
